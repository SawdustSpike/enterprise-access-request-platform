# Enterprise Access Request Platform

A Java 21 / Spring Boot enterprise access governance application that supports access request creation, approval workflows, role-based authorization, audit tracking, filtering, paging, Swagger/OpenAPI documentation, automated tests, and Dockerized local development.

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Maven
- Swagger / OpenAPI
- Docker / Docker Compose
- JUnit / Mockito
